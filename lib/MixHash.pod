=TITLE class Mix

    class MixHash does Mixy { }

A mutable collection of weighted values. The weights are accessable
via the C<{ }> postcircumfix. For an immutable collection, see L<Mix>.

    my $letter-prob = MixHash.new-fp('a' => 1.3, 'b' => 4.8', 'c' => 8);
    $letter-prob<d> = 1.6;
